# Sample speech dataset attribution

The two files are 16 kHz and 8 kHz variants of the same speech recording from
the VOiCES dataset. They are distributed under the Creative Commons
Attribution 4.0 International license (CC BY 4.0).

PyTorch hosts these copies for its TorchAudio tutorials:

- `Lab41-SRI-VOiCES-src-sp0307-ch127535-sg0042.wav`
- `Lab41-SRI-VOiCES-src-sp0307-ch127535-sg0042-8000hz.wav`

Source documentation:
https://docs.pytorch.org/audio/master/tutorials/audio_feature_augmentation_tutorial.html

License:
https://creativecommons.org/licenses/by/4.0/

This tiny fixture is intended for local functional testing. It is not a
representative benchmark dataset, and both rows contain the same utterance.
